# Elite Bank

A full-stack digital banking application built using React.js, Spring Boot, Hibernate, and MySQL for managing accounts, transactions, and banking operations.

## Features

- User Account Management
- Deposit Money
- Withdraw Money
- Fund Transfer Between Accounts
- Transaction History Tracking
- Banking Analytics Dashboard
- Responsive Banking UI
- REST API Integration
- Real-Time Data Updates

## Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- Axios
- CSS3

### Backend
- Spring Boot
- Spring Data JPA
- Hibernate
- REST APIs

### Database
- MySQL

### Tools
- Git & GitHub
- Postman
- Maven
- VS Code

## Architecture

Frontend (React.js)
↓
Axios API Calls
↓
Spring Boot REST APIs
↓
Service Layer
↓
Repository Layer (JPA)
↓
MySQL Database

## Learning Outcomes

- Developed RESTful APIs using Spring Boot.

- Integrated React frontend with backend services using Axios.
- Applied layered architecture (Controller-Service-Repository).
- Managed database operations using MySQL.
